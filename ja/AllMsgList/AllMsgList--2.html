# Profile Configuration Messages

メッセージ 説明
Profile Inquiry 相手機器がサポートしているProfileのリストを要求します。
Reply to Profile Inquiry サポートしているProfileのリストとそのProfileの有効・無効状況を通知します。
Set Profile On 相手機器の無効になっているProfileを有効にします。
Set Profile Off 相手機器の有効になっているProfileを無効にします。
Profile Enabled Report Profileが有効になった場合に通知します
Profile Disabled Report Profileが無効になった場合に通知します
Profile Added Report サポート可能なProfileが増えた場合に通知します。
Profile Removed Report サポート可能なProfileが減った場合に通知します。
Profile Details Inquiry Profileの詳細なサポート情報を要求します。
Reply to Profile Details Inquiry Profileの詳細なサポート情報を返します。
Profile Specific Data Profile固有の情報をやりとりするためのSysExメッセージです。
