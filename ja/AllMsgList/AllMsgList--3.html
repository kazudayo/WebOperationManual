# Property Exchange Messages

メッセージ 説明
Inquiry: Property Exchange Capabilities 相手機器がサポートしているPEの同時リクエスト数やバージョンを問い合わせます。
Reply to Property Exchange Capabilities PEの同時リクエスト数やバージョンを返します。
Inquiry: Get Property Data 相手機器のProperty情報を要求します。
Reply to Get Property Data Property情報を返します。
Inquiry: Set Property Data Property情報の変更を要求します。
Reply to Set Property Data Property情報の変更が完了したことを通知します。
Subscription 相手機器の任意のPropertyに変化があった場合、随時通知するように要求します。
Reply to Subscription 指定されたPropertyが変化した場合に、変化内容を通知します。
Notify PEにおけるエラーやその他情報を通知します。
