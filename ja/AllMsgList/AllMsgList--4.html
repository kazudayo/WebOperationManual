# Process Inquiry

メッセージ 説明
Inquiry: Process Inquiry Capabilities 相手機器のProcess Inquiryサポート状況を要求します。
Reply to Process Inquiry Capabilities Process Inquiryサポート状況を返します。
Inquiry: MIDI Message Report 相手機器のControlChangeやProgramChange、NoteOn/Offの現在状態を問い合わせます。
Reply to MIDI Message Report ControlChangeやProgramChange、NoteOn/Offの現在状態を返します。
End of MIDI Message Report MIDI Message Report通信の完了を通知します。
