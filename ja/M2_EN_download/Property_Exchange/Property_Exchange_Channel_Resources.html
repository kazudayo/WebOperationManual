# Property Exchange Channel Resources

M2-108-UM_v1-01_Channel_Resources.pdf

このドキュメントでは、3 つのプロパティ交換リソースを定義します： ChannelMode、BasicChannelRx、BasicChannelTx です。

これらのリソースは、デバイスがアクティブに使用している MIDI チャンネルの選択に関連する情報を取得および設定するために使用されます。

MIDIチャンネルの使用に対応するために定義された追加のProperty Resource、ChannelListがあり、これはこの文書では定義されていませんが、MIDI-CI Property Exchange Foundational Resourcesにあります： DeviceInfo、ChannelList、JSONSchema にあります。 MIDIチャンネルを複雑に使用するデバイスは、ChannelListリソースも実装する必要があります。
