# 3. MIDI 2.0 ロゴ ライセンス申請と取得

# 手順 1- MIDI 2.0 ロゴ認定要件の確認

# 1-1 ウェブ・ブラウザにて midi.org へアクセスし、[Login] ボタンより、ログインします。

まだ、登録が済んでない場合、 [Register] ボタンより、必要事項を記入の上、登録を完了してからログインしてください。

# 1-2 midi.org/xxx から、次の４つのドキュメントをダウンロードします。

  1. MMA/AMEI Logo Usage Guidelines
    MIDI 2.0 ロゴのサイズ規定や使い方が記述されています。

  2. MMA/AMEI Licensed Mark(s) Requirements
    MIDI 2.0 ロゴを付与できる要件が記述されています。

  3. MMA/AMEI MIDI 2.0 TRADEMARK LICENSE AGREEMENT
    MIDI 2.0商標使用許諾契約書です。

  4. MIDI 2.0 Logo Implementation Checklist
    対象機器の MIDI 2.0 実装内容を記載します。

# 1-3 次の２つのドキュメントを参照して、対象機器がMIDI 2.0ロゴの要件を満たすかを確認します。

1）MMA/AMEI Licensed Mark(s) Requirements
2）MMA/AMEI MIDI 2.0 TRADEMARK LICENSE AGREEMENT
要件を満たす場合、手順２へ進んでください。

# 手順 2- MIDI 2.0商標使用許諾契約書への署名と申請

# 2-1　電子署名の準備

契約書への署名は、次の電子署名のいずれかを利用します。

  • Acrobat Sign
  • DocuSign
    電子署名の宛先には、info@midi.org を指定してください。

# 2-2　MIDI 2.0商標使用許諾契約書に署名

MMA/AMEI MIDI 2.0 TRADEMARK LICENSE AGREEMENT にある、各条項を確認します。
確認後、次の位置に電子署名してください。

図 MIDI 2.0商標使用許諾契約書 記入例

# 2-3　MIDI Association（MIDI 協会）へ送付

MIDI 2.0 ロゴ使用許諾契約書に電子署名すると、電子署名システムにより、MIDI Association（MIDI 協会）に送信されます。
また、次のように、別途、電子メールを送信しておくと安心です。
宛先：　info@midi.org
件名：　[御社名] MIDI 2.0 TRADEMARK LICENSE AGREEMENT
本文：（例）You will receive an email requesting a digital signature with an instruction.Please 　　　　　　kindly review the agreement and proceed the steps of signing.

# 2-4 MIDI 2.0商標使用許諾通知

MIDI Association（MIDI 協会）が、MIDI 2.0 ロゴ使用許諾契約を締結すると、電子署名システムから電子メールにて通知を受けます。
DocuSign の場合、 ”[Docusign] 署名進捗確認用” という件名で、登録した電子メールアドレスに、サイン済の PDF ファイルが添付されて届きます。
MIDI 2.0 ロゴ使用許諾契約書は、大切に保管してください。

# 手順 3- MIDI 2.0機器の認定

MIDI 2.0商標使用許諾契約が締結されていると、実装確認プロセスを開始できます。
次のMIDI 2.0仕様書、及び MIDI 2.0検証用ソフトウェアを利用して、対象機器の実装内容、及び動作を確認してください。

  • MIDI 2.0仕様書 (opens new window)
  • MIDI 2.0検証用ソフトウェア：MIDI Workbench（※リンクと使い方説明ビデオが必要）
  • 実装ガイドライン (opens new window)

# 手順 4- MIDI 2.0実装チェックリストの申請

# 4-1　MIDI 2.0 ロゴ要件を確認します

対象機器が 「MMA/AMEI Licensed Mark(s) Requirements」に準拠していることを確認します。

# 4-2　MIDI 2.0実装チェックリストの記入

「MIDI 2.0 Logo Implementation Checklist」 の各項目を記入します。
MIDI Workbench を利用すると、接続した対象機器の実装状況に応じて、一部の項目を自動設定してくれるので便利です。
各項目を確認し、PDF形式でファイル出力されます。

# 4-3 MIDI 2.0実装チェックリストの申請

各項目を記入した 「MIDI 2.0 Logo Implementation Checklist」 を電子メールで送信します。
宛先：MIDILogoSubmission@midi.org
件名：[御社名] MIDI 2.0 Logo Implementation Checklist
本文：（例）xxxxx

# 4-4 MIDI 2.0機器のロゴ認定の獲得

MIDI Association から、認定結果が送られてきます。
これらの認定プロセスが完了すると、署名済みライセンス契約の条項に従って、MIDI 2.0ロゴマークが使用可能になります。
