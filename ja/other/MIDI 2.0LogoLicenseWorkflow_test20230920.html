# MIDI 2.0 Logo License 認定

# 認定プロセス

認定には、2つのステップがあります。

  1. 企業単位の　MIDI 2.0商標使用許諾 契約
  2. 製品単位の　MIDI 2.0実装チェックリストの提出

  • MIDI 2.0商標使用許諾契約が済んでない場合 手順１からお進みください。

  • MIDI 2.0商標使用許諾契約が済んでいる場合 手順３からお進みください。

    Flow Chart図

# MIDI 2.0 Logo License 申請と取得

# 手順 1- 製品がMIDI 2.0 ロゴの対象であることを確認します

# 1-1 XXX から　次の４つのドキュメントをダウンロードします

  1. MMA/AMEI Logo Usage Guidelines ロゴのサイズ規定や使い方が記述されています

  2. MMA/AMEI Licensed Mark(s) Requirements MIDI 2.0 ロゴを付与できる要件が記述されています

  3. MMA/AMEI MIDI 2.0 TRADEMARK LICENSE AGREEMENT MIDI 2.0商標使用許諾契約書

  4. MIDI 2.0 Logo Implementation Checklist 対象製品の MIDI 2.0 実装内容を記載します

# 1-2 次の２つのドキュメントを参照して、対象製品がMIDI 2.0ロゴの対象となるかを確認します

  1. MMA/AMEI Licensed Mark(s) Requirements
  2. MMA/AMEI MIDI 2.0 TRADEMARK LICENSE AGREEMENT

対象であれば手順２へ進んでください。

# 手順 2- MIDI 2.0商標使用許諾契約書に署名して返送します

# 2-1　電子署名

Adobe Sign 或いは、docusign を利用する準備をしてください。

# 2-2　MIDI 2.0商標使用許諾契約書に署名

MMA/AMEI MIDI 2.0 TRADEMARK LICENSE AGREEMENT にある、各条項を確認します。 次の位置に電子署名してください。

図　

# 2-3　MIDI Association（MIDI 協会）へ送付

署名したMIDI 2.0 ロゴ使用許諾契約書を MIDI Association（MIDI 協会）宛てに電子メール送信する 電子メールアドレスは、 info@midi.org です

# 2-4 MIDI 2.0商標使用許諾通知

MIDI Association（MIDI 協会）からMIDI 2.0商標使用許諾通知を受け取ります。

# 手順 3- 製品毎の認定

MIDI 2.0商標使用許諾契約が締結されていれば、認定プロセスを開始できます。

次のMIDI 2.0仕様書、及び MIDI 2.0ツールを利用して対象製品の動作を確認してください。

  • MIDI 2.0の仕様書へのリンク
  • Amenote protozoa MIDI 2.0プロトタイピングボード
  • MIDI Workbench
  • 実装ガイドライン

（MIDI 2.0製品のテストは各メーカーの責任で行いますが、MIDI Association では常に質問への答えを提供しています）

# 手順 4- MIDI 2.0実装チェックリストを完成させて送信します

# 4-1　MIDI 2.0 ロゴ要件を確認します

対象製品が 「MMA/AMEI Licensed Mark(s) Requirements」に準拠していることを確認します

# 4-2　MIDI 2.0 Logo Implementation Checklist

「MIDI 2.0 Logo Implementation Checklist」 の各項目を記入します。 MIDI Workbench を利用して、「Implementation Checklist base（仮）」を出力してから、各項目を確認するのが効率的です。

# 4-3 チェックリストを電子メールで送信します

「MIDI 2.0 Logo Implementation Checklist」 のコピーを電子メールで送信します 送信先メールアドレスは、MIDILogoSubmission@midi.org です

# 4-4 ロゴライセンス認定

MIDI Association から、認定結果が送られてきます。 これらの手順が完了すると、「MIDI 2.0 Logo Implementation Checklist」 を提出した製品、および署名済みライセンス契約の条項に従って、MIDI 2.0ロゴマークを使用するライセンスが付与されます。

  • MIDI 2.0ロゴの運用については、「MMA/AMEI Logo Usage Guidelines」、及び「ロゴ運用ガイドライン補足（仮）」をご確認ください。
  • MIDI 2.0ロゴ電子データは、XXXからダウンロードしてください。（アクセス制限）

# 手順 ５- 製品の公開通知（option）

対象製品が一般公開、或いは発表されていること（または解禁日を記載した製品リリース情報）を MIDI Association に電子メールにて連絡してください。 電子メールアドレスは、 info@midi.org です

この連絡により、MIDI Association にて、MIDI 2.0製品リストに追加、管理を行います。 その後、midi.org WebPage に情報掲載します。
