# メッセージリスト

# メッセージタイプ カテゴリー メッセージ ひとこと説明 Transaction別
1 MIDI-CI Management Messages Discovery MIDIネットワーク上のMIDI-CI機器を探します。 1
2 MIDI-CI Management Messages Reply to Discovery Discoveryメッセージに対する応答です。 1
3 MIDI-CI Management Messages Inquiry: Endpoint Information 相手機器のEndpoint情報を要求します。 2
4 MIDI-CI Management Messages Reply to Endpoint Information Endpoint情報を返します。 2
5 MIDI-CI Management Messages MIDI-CI ACK 処理中の機器がタイムアウトを防ぐために送ります。 3
6 MIDI-CI Management Messages MIDI-CI NAK MIDI-CIの通信に失敗したことを通知します。 3
7 MIDI-CI Management Messages Invalidate MUID MUIDが無効になった場合の通知です。 4
8 MIDI-CI Profile Configuration Messages Profile Inquiry 相手機器がサポートしているProfileのリストを要求します。 5
9 MIDI-CI Profile Configuration Messages Reply to Profile Inquiry サポートしているProfileのリストとそのProfileの有効・無効状況を通知します。 5
10 MIDI-CI Profile Configuration Messages Set Profile On 相手機器の無効になっているProfileを有効にします。 6
11 MIDI-CI Profile Configuration Messages Set Profile Off 相手機器の有効になっているProfileを無効にします。 6
12 MIDI-CI Profile Configuration Messages Profile Enabled Report Profileが有効になった場合に通知します 6
13 MIDI-CI Profile Configuration Messages Profile Disabled Report Profileが無効になった場合に通知します 6
14 MIDI-CI Profile Configuration Messages Profile Added Report サポート可能なProfileが増えた場合に通知します。 7
15 MIDI-CI Profile Configuration Messages Profile Removed Report サポート可能なProfileが減った場合に通知します。 7
16 MIDI-CI Profile Configuration Messages Profile Details Inquiry Profileの詳細なサポート情報を要求します。 8
17 MIDI-CI Profile Configuration Messages Reply to Profile Details Inquiry Profileの詳細なサポート情報を返します。 8
18 MIDI-CI Profile Configuration Messages Profile Specific Data Profile固有の情報をやりとりするためのSysExメッセージです。 9
19 MIDI-CI Property Exchange Messages Inquiry: Property Exchange Capabilities 相手機器がサポートしているPEの同時リクエスト数やバージョンを問い合わせます。 10
20 MIDI-CI Property Exchange Messages Reply to Property Exchange Capabilities PEの同時リクエスト数やバージョンを返します。 10
21 MIDI-CI Property Exchange Messages Inquiry: Get Property Data 相手機器のProperty情報を要求します。 11
22 MIDI-CI Property Exchange Messages Reply to Get Property Data Property情報を返します。 11
23 MIDI-CI Property Exchange Messages Inquiry: Set Property Data Property情報の変更を要求します。 12
24 MIDI-CI Property Exchange Messages Reply to Set Property Data Property情報の変更が完了したことを通知します。 12
25 MIDI-CI Property Exchange Messages Subscription 相手機器の任意のPropertyに変化があった場合、随時通知するように要求します。 13
26 MIDI-CI Property Exchange Messages Reply to Subscription 指定されたPropertyが変化した場合に、変化内容を通知します。 13
27 MIDI-CI Property Exchange Messages Notify PEにおけるエラーやその他情報を通知します。 14
28 MIDI-CI Process Inquiry Inquiry: Process Inquiry Capabilities 相手機器のProcess Inquiryサポート状況を要求します。 15
29 MIDI-CI Process Inquiry Reply to Process Inquiry Capabilities Process Inquiryサポート状況を返します。 15
30 MIDI-CI Process Inquiry Inquiry: MIDI Message Report 相手機器のControlChangeやProgramChange、NoteOn/Offの現在状態を問い合わせます。 16
31 MIDI-CI Process Inquiry Reply to MIDI Message Report ControlChangeやProgramChange、NoteOn/Offの現在状態を返します。 16
32 MIDI-CI Process Inquiry End of MIDI Message Report MIDI Message Report通信の完了を通知します。 16
33 UMP MT F UMP Stream Endpoint Discovery UMP Stream上のMIDI Endpointを探します。 17
34 UMP MT F UMP Stream Endpoint Info Notification Endpoint Discoveryメッセージに対する応答です。 17

36 UMP MT F UMP Stream Endpoint Name Notification MIDI Endpoint名を通知します。 17
37 UMP MT F UMP Stream Product Instance ID Notification 製品のインスタンスID(シリアル番号)を通知します。 17

40 UMP MT F UMP Stream Function Block Discovery 相手Endpointに対して利用可能なFunction Blockの情報を要求します。 18
41 UMP MT F UMP Stream Function Block Info Notification Function Block情報を返します。 18
42 UMP MT F UMP Stream Function Block Name Notification Function Block名を返します。 18
43 UMP MT F UMP Stream Start of Clip MIDI Clipの再生開始時に最初に送るメッセージです。 19
44 UMP MT F UMP Stream End of Clip MIDI Clipの再生終了時に最後に送るメッセージです。 19
