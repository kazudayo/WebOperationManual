# MIDI-CI Specification

MIDI-CI: 後方互換性を保護しながらMIDIを拡張するソリューション:

MIDI Capability Inquiry (MIDI-CI)は、新しく定義された機能を理解しないMIDI機器との後方互換性を保護しながら、新しい機能によるMIDIの拡張を可能にするメカニズムです。

MIDI-CIは、古いMIDI製品と新しい機能を持つ製品を分離し、2つのMIDIデバイスがどのような新しい機能がサポートされているかを理解するためのメカニズムを提供します。MIDI-CIは双方向通信を前提とし、それを必要とします。

MIDI-CI接続がデバイス間で確立されると、クエリメッセージとレスポンスメッセージによって各デバイスが持つ機能が定義されます。そしてMIDI-CIは、デバイス間で共通の機能を使用するようにネゴシエートまたは自動設定を行います。

MIDI-CIには、MIDI機能を拡張する3つの主要分野のクエリが含まれています:

  1. Profile Configuration

  2. Property Exchange

  3. Process Inquiry
